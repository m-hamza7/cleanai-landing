# How to use this vault

1. Open **Obsidian**
2. **Open folder as vault** → select  
   `cleanai-landing/obsidian`
3. Start at [[00 Home]]
4. Use **Graph view** to browse colored folder groups
5. Follow `[[wikilinks]]` between notes
6. Put temporary notes in `Inbox/`, media in `Attachments/`

## Conventions

- Note titles are unique and match filenames (Obsidian default)
- YAML frontmatter holds `tags` (and sometimes `aliases`)
- Prefer linking to vault notes over duplicating root `*.md` content
- Never paste production secrets into notes — list **variable names** only ([[Environment Variables]])
